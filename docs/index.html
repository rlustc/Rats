Reward-Aware Trajectory Shaping for Few-step Visual Generation

RATS aligns few-step visual generators with preference rewards by shaping intermediate denoising trajectories, using teacher guidance only when it improves reward-aware training.

Rui Li*, Bingyu Li*, Yuanzhi Liang, Haibin Huang, Chi Zhang, XueLong Li

Code Paper BibTeX

Comparison with teacher imitation and terminal reward optimization paradigms.

Abstract

Few-step generation Preference alignment Trajectory distillation Reward gating

Achieving high-fidelity generation in extremely few sampling steps has long been a central goal of generative modeling. Existing distillation methods compress multi-step denoising into a few-step generator, but often constrain the student to imitate a stronger teacher. RATS introduces preference alignment awareness through horizon-matched teacher-student latent trajectories and a reward-aware gate that adaptively regulates teacher guidance based on relative reward performance. The teacher is used only during training, preserving the deployment efficiency of the few-step student.

Method Overview

RATS treats few-step alignment as trajectory-level reward allocation rather than only output-level reward optimization. The framework combines multi-stage latent alignment, sigma-based horizon matching, and reward-gated teacher guidance.

Horizon matching

Aligns teacher and student latent trajectories at key denoising stages, transferring intermediate priors from a stronger multi-step generator.

Reward-aware gate

Strengthens trajectory shaping when the teacher receives higher rewards, and relaxes the teacher constraint when the student catches up or surpasses it.

No test-time overhead

Uses the EMA teacher only during training; inference keeps the original few-step student speed and deployment footprint.

Visual Results

Experiments show that RATS improves the efficiency-quality trade-off for few-step visual generation, narrowing the gap between few-step students and stronger multi-step generators.
