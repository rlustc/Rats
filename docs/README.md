# RATS Homepage

This directory contains the static GitHub Pages homepage for **Reward-Aware Trajectory Shaping for Few-step Visual Generation**.

To publish it from the existing `rlustc/Rats` repository:

1. Push this `docs/` directory to the default branch.
2. Open `Settings` -> `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select the default branch and `/docs`.
5. Save and wait for the deployment to finish.

The project site URL should be:

```text
https://rlustc.github.io/Rats/
```

For a GitHub Organization homepage instead, create a repository named `<organization>.github.io`, put these files at that repository root, and publish from the root folder.
